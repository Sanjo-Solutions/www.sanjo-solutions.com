# Webseite erstellen

Sie können eine Webseite mit [AWS Lightsail](https://aws.amazon.com/lightsail/). Wenn Sie eine Instanz erstellen, wählen Sie "Linux" als Plattform und "WordPress" als Plan aus.

Falls Sie hilfe brauchen, können Sie mich [kontaktieren](/de/kontaktieren/).
