# How to install rsync into Git bash

Download [rsync](https://repo.msys2.org/msys/x86_64/rsync-3.2.7-2-x86_64.pkg.tar.zst).

Download [lib-xxhash](https://repo.msys2.org/msys/x86_64/libxxhash-0.8.2-1-x86_64.pkg.tar.zst).

Note: The latest versions can be found via the list at [https://repo.msys2.org/msys/x86_64/](https://repo.msys2.org/msys/x86_64/).

Extract `rsync.exe` (which is inside the `usr/bin` folder of the downloaded rsync archive) into `C:\Program Files\Git\usr\bin`.

Extract `msys-xxhash-0.dll` (which is inside the `usr/bin` folder of the downloaded lib-xxhash archive) into `C:\Program Files\Git\usr\bin`.
